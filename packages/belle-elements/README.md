## Install

> npm install @belleui/belle-elements

[](#useage)

## Useage

```javascript
import { LitElement, html } from 'lit-element'
import '@belleui/belle-elements'

class MyApp extends LitElement {
  render() {
    return html`
      <belle-button type="primary">Button</belle-button>
    `
  }
}
```
